# Django i-Shop
  Online store on Python
### 1. Installation

```bash
$ git clone https://github.com/oleg-soshko/django_store.git
$ cd mysite
$ pip install -r requirements.txt
```

### 2. Running The App

```bash
python manage.py runserver
```

### 3. Viewing The App

Go to `http://127.0.0.1:5000`

### 4. Technology Stacks
- Python
- Django
- HTML
- CSS
- bootstrap
